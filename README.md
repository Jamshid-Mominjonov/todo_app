# ToDo App

A modern **ToDo mobile application** built with **Flutter**.  
The app helps users manage daily tasks efficiently with offline support.

---

## ✨ Features

- 📝 **CRUD operations**: create, read, update, and delete tasks
- 📦 **Offline storage** with Hive DB

---

## 🛠️ Technologies

- **Flutter** (Dart)
- **Provider** (State Management)
- **Hive DB** (local database)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Jamshid-Mominjonov/todo_app.git
   
2. **Install dependencies**
    ```bash
    flutter pub get
    ```

3. **Run the app**
    ```bash
    flutter run
    ```
